# Changelog

All notable changes will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/) and the project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2025-11-21
### Added
- Workbook-first schemagen and import workflow for SQL Server.
- Configurable column naming styles with CLI overrides and config fallbacks.
- `--write-db`, `--ensure-table`, and multiple write modes (insert, truncate-insert, upsert).
- Validation pipeline with JSON export of valid rows.
- Comprehensive CLI documentation, CONTRIBUTING guide, and MIT license.
